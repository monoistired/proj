Zelda

Game Information

Game Name: Zelda

Region: USA

Console: SNES

Game Release: 1992-04-13 (32 years ago)

Genre: Action RPG, Hack

Publisher: N

Languages: English

Image Format: SFC/SMC

Description:

This is a romhack of The Legend of Zelda: A Link to the Past. It changes lots of things including link's and other characters sprites colors to resemble the original artwork. Renames several items like the 'Flute' to 'Ocarina' (it's an Ocarina). Reverts censored graphics back from the JP release (Star of David, Skulls statues, Sanctuary doorway, etc). Increase max consumables (bombs, arrows) to 99. And a lot more see included readme or the link to the patch below.
